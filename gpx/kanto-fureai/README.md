# Kanto Fureai no Michi

This directory holds GPX files for the Kanto Fureai no Michi
(関東ふれあいの道).

The full route is divided across multiple courses spanning seven regions. Links
to each regions site are below.

1. [Tokyo](https://www.kankyo.metro.tokyo.lg.jp/nature/nature_park/kanfure)
1. [Gunma](https://www.pref.gunma.jp/page/1172.html)
1. [Saitama](https://www.pref.saitama.lg.jp/a0508/shisetsu/kanfuretop.html)
1. [Kanagawa](https://www.pref.kanagawa.jp/docs/f4y/02yama/choukyori/kantou01.html)
1. [Tochigi](https://www.pref.tochigi.lg.jp/d04/intro/shizen/kouen/kantoindex.html)
1. [Ibaraki](https://www.pref.ibaraki.jp/seikatsukankyo/shizen/shizen/kanto-fureai.html)
1. [Chiba](https://www.pref.chiba.lg.jp/shizen/choubou/fureai/index.html)
